Variaveis

Variáveis funcionam como um repositório onde vc pode armazenar valores, dados, etc

Tipos de variáveis: string / number / boolean

Regras de declaração:

  • Iniciadas apenas com letras ou _
  • Não podem ter caracteres especiais
  • Não podem ser iguais as palavras reservadas da linguagem
  • Varáveis são case sensitive

Declaração:

A declaração começa com a palavra reservada var seguida pelo nome da variável depois o sinal de = (igual) seguido pelo valor que deseja atribuir a variável, como abaixo

var SenhaDoPc = 56155
