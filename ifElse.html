If Else

if else é uma estrutura de controle, podemos fazer verificações antes da pagina tomar um determinado fluxo e mudalo, se for preciso

estrutura

  • if(condição){
  •     codigo a ser executado
  • }else{
  •     codigo a ser executado
  • }

Se a condição for verdadeira o bloco if é executado, se não for o do else é executado, o else pode ser omitido se não for nescessário

também é possivel encadear varias condições, como abaixo

  • if(condição){
  •     codigo a ser executado
  • }else if(condição){
  •     codigo a ser executado
  • }else{
  •     codigo a ser executado
  • }
