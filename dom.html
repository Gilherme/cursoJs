DOM - Document Object Model

Em js usamos o DOM para manipolar elementos da tela, movimentar, esconder, fazer lógica para saber a ordem deles, etc.

Selecionando elementos

As quatro maneiras de selecionar um elemento mais comuns e que com certeza você vai usar, são:

  • getElementById() Seleciona o elemento pelo Id
  • getElementsByTagName(): Seleciona o elemento pelo Id
  • getElementsByClassName(): Seleciona o elemento pelo Id
  • getElementsByName(): Seleciona o elemento pelo Id

Recuperando valores dos elementos

  • document.getElementById().value Recupera o valor do elemento, em input, recupera o valor digitado, por exemplo
  • document.getElementById().style.background Recupera a propriedade background (poder usar outras propriedades)
