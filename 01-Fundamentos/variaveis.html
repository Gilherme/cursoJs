Variaveis

Variáveis funcionam como um repositório onde vc pode armazenar valores, dados, etc

Tipos de variáveis: string / number / boolean

Regras de declaração:

  • Iniciadas apenas com letras ou _
  • Não podem ter caracteres especiais
  • Não podem ser iguais as palavras reservadas da linguagem
  • Varáveis são case sensitive

Declaração:

A declaração começa com a palavra reservada var seguida pelo nome da variável depois o sinal de = (igual) seguido pelo valor que deseja atribuir a variável, como abaixo

var SenhaDoPc = 56155

Concatenação

Usando variavéis e possivel combinalas (concatenar) com outros blocos de código, string, etc

Para concatenar uma frase com uma variavel usamos o operador +

Por exemplo document.write('Olá ' + nome + ' Estamos felizes em ter você aqui')

No exemplo acima usamos uma das Funções Nativas document.write() Para imprimir no body uma saudação com o nome dinamico, assim, quando a pagina reconhecer o usuario, a variavel 'nome' passa a ser automaticamente o nome dele

IMPORTANTE - Se tentarmos somar duas strings numericas, por exemplo, document.write( '20' + '50') o resultado impresso será 2050, ou seja, ele irá concatenar os numeros

Usando a Função nativa alteramos o valor numerico no formato string para o formato number
